# 📡 Socket.IO Events

| Event | Description |
|------|-------------|
| `connection` | User connects |
| `sendMessage` | Send new message |
| `receiveMessage` | Receive message instantly |
| `userOnline` | User active status |
| `disconnect` | User disconnected |
